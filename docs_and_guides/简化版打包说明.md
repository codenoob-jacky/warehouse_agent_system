# 简化版一键打包说明

## 🎯 打包脚本说明

我创建了两个打包脚本，解决了编码问题：

### 1. `build_simple.bat` (英文版)
- 全英文界面，避免编码问题
- 适合国际化环境
- 生成英文版用户界面

### 2. `一键打包简化版.bat` (中文版)
- 使用 `chcp 65001` 设置UTF-8编码
- 中文界面，适合国内用户
- 生成中文版用户界面

## 🔧 编码问题解决

### 问题原因
- Windows批处理文件默认使用GBK编码
- PyInstaller处理中文路径和文件名容易出错
- 不同系统的编码设置不一致

### 解决方案
1. **设置UTF-8编码**: `chcp 65001 >nul`
2. **使用英文文件名**: 避免中文路径问题
3. **静默安装**: 使用 `-q` 参数减少输出
4. **错误处理**: 提供备用安装源

## 🚨 注意事项

1. **网络连接**: 需要网络连接进行OCR识别
2. **端口占用**: 确保8000端口未被占用
3. **防火墙**: 可能需要允许程序通过防火墙
4. **系统兼容**: 支持Windows 7/8/10/11

## 📞 故障排除

### 打包失败
- 检查Python版本（需要3.8+）
- 尝试使用管理员权限运行
- 检查网络连接（下载依赖需要）

### 运行失败
- 检查端口8000是否被占用
- 确认网络连接正常
- 查看防火墙设置

### OCR识别失败
- 检查网络连接
- 确认OCR.Space服务可访问
- 尝试更换API密钥

这样的设计确保了打包过程的稳定性和用户体验的一致性！
